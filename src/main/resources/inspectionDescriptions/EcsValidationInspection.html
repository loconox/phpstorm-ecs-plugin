Reports coding style violations detected by Easy Coding Standard (ECS) in PHP files.

This inspection runs ECS in the background and highlights lines that do not comply with the coding standards defined in your ECS configuration file (e.g. ecs.php). Violations are shown as warnings directly in the editor, giving you immediate feedback while you write code.

The plugin also integrates with the Reformat Code action, allowing ECS to automatically fix coding style issues when you format your file.

Configure the path to the ECS binary and the configuration file in Settings | PHP | Quality Tools | Easy Coding Standard.
